# DAY1: 2024年7月28日(日)

# 移動

きっぷの受取には、「受取コード」と「パスワード」が必要です。

```
乗車日　2024年7月28日
新大阪(09:06)→のぞみ4号→名古屋(09:55)
N700S系16両　
おとな1名  こども1名
スマートEX
発売額　10,010円
```

- 地下鉄の場合
    - 科学館最寄りの伏見駅へは名古屋駅から地下鉄東山線、藤が丘行きで1駅
    - 伏見駅からは4・5番出口を出て科学館まで徒歩600m
- 荷物多いし名古屋駅からタクシーがいいかも

# [名古屋市科学館](http://www.ncsm.city.nagoya.jp/)

- 月曜休館
- 開館時間 9:30～17：00
- オンライン入館券あり
